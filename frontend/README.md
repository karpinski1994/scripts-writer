# Scripts Writer — Frontend

Next.js frontend for the Scripts Writer application.

## Quick Start

For a complete setup guide including the backend and dev script, see the [root README](../README.md).

```bash
# From the frontend/ directory
npm install
npm run dev
```

Or use the root `dev.sh` script to run both backend and frontend:

```bash
../dev.sh start
```

The frontend runs on **http://localhost:5173** (or check the dev.sh output for the actual port).

## Development

```bash
npm run dev      # Start development server
npm run build    # Production build
npm run start    # Start production server
npm run lint     # Run ESLint
```
